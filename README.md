# Ubuntu Book — Guia Completo

  Material definitivo sobre Ubuntu em Português Brasileiro. Do iniciante ao avançado.

  ## 🌐 Site

  Acesse em: **https://wallysondevs.github.io/ubuntu-book/**

  ## 📚 Conteúdo

  - História e Filosofia do Ubuntu
  - Guia de Instalação e Primeiros Passos
  - GNOME e Ambiente Gráfico
  - Gerenciamento de Pacotes (APT, Snap, Flatpak)
  - Systemd e Serviços
  - Sistema de Arquivos e Navegação
  - Permissões, Usuários e Processos
  - Redes, SSH e Segurança
  - Shell Bash e Scripting
  - Troubleshooting

  ## 🛠️ Desenvolvimento Local

  ```bash
  git clone https://github.com/Wallysondevs/ubuntu-book.git
  cd ubuntu-book
  npm install
  npm run dev
  ```

  ## 🚀 Deploy

  O site é automaticamente publicado no GitHub Pages a cada push na branch `main`.
